Labama Hotel AI Assistant

An AI-powered hotel assistant built with LangChain, OpenAI, Gradio, and SQLite.
It can answer hotel-related questions, check room availability, and handle customer complaints.

Features

Conversational hotel chatbot using LangChain.

SQL-based room availability and price checking.

General hotel info retrieval using RAG.

Complaint detection with push notifications.

Simple Gradio chat interface.
